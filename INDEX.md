# Prompt Engineering Library Index

This is a comprehensive index of all techniques and examples in this library. Use this to navigate and find what you're looking for.

## Techniques Overview

| # | Technique | Description | Complexity | Best For |
| :--- | :--- | :--- | :--- | :--- |
| **01** | Zero-Shot Prompting | Basic prompting without examples | Low | Baseline, quick answers |
| **02** | Few-Shot Prompting | Providing examples to guide the AI | Low-Medium | Format consistency, style guidance |
| **03** | Chain-of-Thought | Asking the AI to think step-by-step | Medium | Strategic thinking, complex tasks |
| **04** | Persona Prompting | Assigning a role to the AI | Low-Medium | Authentic voice, brand alignment |
| **05** | Structured Output | Specifying output format (JSON, CSV, etc.) | Medium | Data integration, automation |
| **06** | Negative Prompting | Telling the AI what NOT to do | Low | Avoiding specific outputs |
| **07** | RAG (Retrieval-Augmented Generation) | Providing external data to the AI | High | Knowledge-grounded responses |
| **08** | Meta-Prompting | Asking the AI to improve its own prompts | High | Iterative refinement, optimization |

## Learning Paths

### Path 1: Beginner (Start Here!)

1.  Read `README.md` to understand the library's purpose.
2.  Read `HOW_TO_USE.md` to learn how to explore.
3.  Explore `/01-zero-shot-prompting` to see basic prompting.
4.  Explore `/02-few-shot-prompting` to see how examples help.
5.  Read `COMPARISON_GUIDE.md` to understand the differences.

### Path 2: Intermediate (Building Skills)

1.  Complete the Beginner path.
2.  Explore `/03-chain-of-thought-prompting` to learn strategic thinking.
3.  Explore `/04-persona-prompting` to learn about brand voice.
4.  Explore `/05-structured-output-prompting` to learn data integration.
5.  Try combining techniques from different sections.

### Path 3: Advanced (Mastery)

1.  Complete the Intermediate path.
2.  Explore `/06-negative-prompting` to learn constraint-based prompting.
3.  Explore `/07-retrieval-augmented-generation` to learn knowledge-grounded prompting.
4.  Explore `/08-meta-prompting` to learn iterative refinement.
5.  Create your own prompts by combining multiple techniques.
